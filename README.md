### Client usage:

- Open Packages/User directory.  Easy way to do this is, selecting Browse Packages in Sublime Text / Preferences.

- Copy into Packages/User directory:
	- LiveCoding.py
	- Main.sublime-menu

### Instructor usage:

- Open Packages/User directory.  Easy way to do this is, selecting Browse Packages in Sublime Text / Preferences.

- Copy into Packages/User directory:
	- LiveCodingInstructor.py
	- Main.sublime-menu

- Download server.go and item.go
- go run *.go
